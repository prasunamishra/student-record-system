# Student Record System

A simple web-based Student Record Management System** built with **PHP, MySQL, HTML, CSS, JavaScript, and AJAX**.

This project allows an administrator to log in and manage student records, courses, modules, attendance, and grades.

##  Live Demo

You can view this project hosted online here:  
 https://student.heraldcollege.edu.np/~np03cs4s250105/student-record-system/public


# Features

- **Admin Login & Logout**
- View and manage students
- View courses and modules
- Track attendance
- Manage grades
- AJAX-powered search for student records
